# Welcome

This book takes you from an empty laptop to a running AI agent you can talk
to, in one sitting. You do not need to know Kubernetes; you will meet the
handful of ideas you need exactly when you need them.

**This book is not written yet.** Agentry v0.1.0 requires building from
source, which is no way to start a tutorial. Writing begins once v0.2.0
ships a one-command install. The chapter list below is the promise of what
is coming.

If you cannot wait and are comfortable building from source, the
[User Guide](https://github.com/win07xp/kubeclaw) covers installation and
first steps today.
