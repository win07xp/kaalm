# Welcome

This book takes you from an empty laptop to a running AI agent you can talk
to, in one sitting. You do not need to know Kubernetes; you will meet the
handful of ideas you need exactly when you need them.

**This book is not written yet.** Kaalm now installs with a single Helm
command, but a tutorial is only as good as the exact commands in it, so this
one is written and walked against a stable release and waits for v0.2.0. The
chapter list below is the promise of what is coming.

To install and use Kaalm today, the
[User Guide](https://github.com/win07xp/kaalm) covers installation and first
steps.
